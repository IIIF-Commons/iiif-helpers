# Official IIIF Presentation 4 Scene fixtures

These files preserve the JSON content of examples from the IIIF Presentation
4.0 Release Candidate source. The checked-in files add one final POSIX newline;
the upstream files at this pin omit it.

- Repository: https://github.com/IIIF/api
- Pinned commit: [`28a88829699ebbbe7722b4692cf3b7b67969bc6c`](https://github.com/IIIF/api/tree/28a88829699ebbbe7722b4692cf3b7b67969bc6c/source/presentation/4.0/example)

Only the two examples used by the narrow Scene and Activation helper tests are
copied here. The tests demonstrate Model paintables, ordered transforms, 3D
point selectors, activating annotations, and ordered actions. They do not
claim rendering support for cameras, lighting, or audio.
